# Loan Processing Co-Pilot

A DSCR (Debt Service Coverage Ratio) loan processing application that helps loan processors manage real estate investment loans.

## Features

- Google Drive integration for document management
- Gmail integration for email communications  
- OpenAI-powered intelligent assistance
- Document tracking and categorization
- Multi-lender support (Kiavi, Roc Capital, AHL, etc.)
- Contact management for borrowers, title agents, insurance agents

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, Shadcn UI
- **Backend**: Node.js, Express.js
- **Database**: PostgreSQL (Neon serverless)
- **ORM**: Drizzle
- **APIs**: Google Drive, Gmail, OpenAI
